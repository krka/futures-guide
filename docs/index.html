Intro

This is an interactive tutorial for futures. The code samples are intended to look and behave like Java but it's actually JavaScript. "->" is automatically converted to "=>" so you can write lambdas like in Java.

Exceptions are similar to Java, but without anything fancy.

The CompletableFuture API you can use here is a partial re-implementation in JavaScript. The functions you can use are listed below:

Since Java 8:

  • var x = new CompletableFuture(); // Create a new future
  • var x = CompletableFuture.completedFuture(value); // Create a new future that is already completed
  • var newFuture = future.thenApply(value -> ...); // Transform a future - the lambda must return a value!
  • var newFuture = future.thenCompose(value -> ...); // Transform a future - the lambda must return a CompletableFuture!
  • var newFuture = future.exceptionally(exc -> ...); // Transform a future - the lambda must return a value!

Since Java 9:

  • var x = CompletableFuture.failedFuture(exception); // Create a new future that is already completed (exceptionally)

Since Java 12:

  • var newFuture = future.exceptionallyCompose(exc -> ...); // Transform a future - the lambda must return a CompletableFuture!
